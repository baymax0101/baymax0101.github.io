print用法

\n 表示换行

变量

变量是用来存储或指代值的，变量不能以数字开头，变量用等号(=)赋值。

变量也可以给变量赋值

变量取名应该易于理解和记忆，Python的变量命名约定为全部小写，不同单词用下划线分割开，例如user_name，原本空格的地方全部用下划线连接，变量名大小写敏感，且不能占据关键字，例如print。

数据类型

整型 a = 1

布尔型 c = True / d = False, 首字母必须大写
字符串型 e = “I love you three thousand “

列表

Python的列表与C语言中的数组类似，但是不限数据元素的类型。一个列表用[]表示,列表是可变的

对象.方法名(…)

列表可以存放不同类型数据, 如果想获得某个位置上的元素，可以使用索引，索引也可更新列表

常用的函数有min(),max(),sorted()

字典

字典用于储存键值对，键用于查找值，字典用{“key”:”value”}表示，键值对之间以逗号分隔，键的类型必须是不可变的。Python提供一个不可变但很像列表的数据结构：元组（tuple），用()表示，元组不能用append和remove等方法，可以用于字典中需要多个值确定一个键的场景，可把整个元组作为一个键。

字典和列表一样是可变的

确定一个键值对是否在字典中，可用 “键” in 字典 确定，返回结果为True或False

面向对象编程

类是创建对象的模板，对象是类的实例。类的三个特点：封装，继承，多态

封装：使用类的人只通过外部接口调用方法，而不知道实现细节

继承：允许创建有层次的类（父类，子类）

多态：同样的接口，依据对象具体类的不同，实现方式也不同

类

Python创建类时使用pascal风格命名，即首字母大写分隔单词，类名后跟冒号

类有一个特殊的方法叫构造函数，主要作用是定义实例对象的属性，必须被命名为_init_(self)：

构造函数第一个参数是一直被占用的，用于表示对象自身，约定成俗称为self。类的属性都写在构造函数中

python的方法第一个参数也为self,后跟传递的参数

类的继承

子类可以用super关键字继承父类的方法

文件操作

读取

open()函数，用于打开目标文件，有三个参数

  1. 文件路径，可以是绝对\ 相对路径
  2. 表示读取模式，可以取”r”或”w”，表示读和写;”a”表示附加内容，会在原文件后面跟上新内容
  3. encoding = “…” 指定以什么编码读取文件

read()函数，用于读取目标文件内容，默认情况下会阅读全部内容并以字符串返回文件内容，同一文件第二次调用read()时就会返回空，目标文件最好是txt格式

readline()函数，一次读取一行，常与while循环搭配使用

readlines()，读取全部内容，并返回每行组成的字符串列表，常与for循环搭配使用

读取完以后要用close()释放资源，也可以在代码前加上with ,后面跟上as f: ,就不需要再close，但原代码要缩进

写入

“w”模式在写以前会把原文件清空，谨慎使用

write()函数可以追加内容，默认不会换行

open()的模式要为”w”或”a”,如果原文件不存在Python会创建一个新文件，这两种模式下均不能读取文件内容；要同时进行读和写，就需要把模式设为”r+”，且write的内容会追加在原文件后

异常处理

通过try/except语句捕捉异常，try后的代码块放容易报错的代码，except后跟异常的名字

except后跟明确的异常类型，如果不知道异常类型就什么都不写

else后跟没有错误时会运行的代码

finally后的内容不管程序错误与否都会执行

测试

assert，即断言，后跟布尔表达式（值为True或False的表达式），出现异常时终止程序运行。

unittest库，用于单元测试，即对软件中最小的可测试单元进行验证，比如验证某函数输出是否符合预期。

在测试类下面写测试用例，每个测试用例是一个方法，要注意测试用例必须以test开头Python才能识别。

为防止assert中断整个程序运行，可以使用assertEqual()方法，括号内跟布尔表达式。

在终端使用python -m unittest，python就会自动执行测试用例。

测试结果有几个点就代表有几个测试用例，点代表测试通过，F表示未通过。
