变量

变量是用来存储或指代值的，变量不能以数字开头，变量用等号(=)赋值。

变量也可以给变量赋值

变量取名应该易于理解和记忆，Python的变量命名约定为全部小写，不同单词用下划线分割开，例如user_name，原本空格的地方全部用下划线连接，变量名大小写敏感，且不能占据关键字，例如print。

字典

字典用于储存键值对，键用于查找值，字典用{“key”:”value”}表示，键值对之间以逗号分隔，键的类型必须是不可变的。Python提供一个不可变但很像列表的数据结构：元组（tuple），用()表示，元组不能用append和remove等方法，可以用于字典中需要多个值确定一个键的场景，可把整个元组作为一个键。

字典和列表一样是可变的

确定一个键值对是否在字典中，可用 “键” in 字典 确定，返回结果为True或False

contact = {"tom":"123456","jerry":"456789"}

面向对象编程

类是创建对象的模板，对象是类的实例。类的三个特点：封装，继承，多态

封装：使用类的人只通过外部接口调用方法，而不知道实现细节

继承：允许创建有层次的类（父类，子类）

多态：同样的接口，依据对象具体类的不同，实现方式也不同

类

Python创建类时使用pascal风格命名，即首字母大写分隔单词，类名后跟冒号

类有一个特殊的方法叫构造函数，主要作用是定义实例对象的属性，必须被命名为_init_(self)：

构造函数第一个参数是一直被占用的，用于表示对象自身，约定成俗称为self。类的属性都写在构造函数中

class CuteCat:
    # 注意构造函数前后的下划线都是两个
    def __init__(self,cat_name,cat_age,cat_color):  
        self.name = cat_name
        self.age = cat_age
        self.color =  cat_color
     
    def speak(self):
        print("喵" * self.age)
     
    def think(self,content):
        print(f"小猫{self.name}在思考{content}...")

cat1 = CuteCat("Tom",2,"orange")

print(f"小猫{cat1.name}的年龄是{cat1.age}岁，花色是{cat1.color}")

类的继承

子类可以用super关键字继承父类的方法

class employee:
    def __init__(self,name,id):
        self.name = name
        self.id = id
        
    def print_info(self):
        print(f"员工姓名:{self.name},员工编号:{self.id}")
        
    def calculate_monthly_pay(self):
        pass
    
    # 子类继承父类，写法是 class B(A):   B是子类，A是父类
class FullTimeEmployee(employee):
    def __init__(self,name,id,monthly_salary):
        super().__init__(name,id)           # 子类可以用super关键字调用父类的构造函数
        self.monthly_salary = monthly_salary
        
    def calculate_monthly_pay(self):
        return self.monthly_salary
    
class PartTimeEmployee(employee):
    def __init__(self,name,id,daily_salary,work_days):
        super().__init__(name,id)
        self.daily_ssalary = daily_salary
        self.work_days = work_days
        
    def calculate_monthly_pay(self):
        return self.daily_salary * self.work_days

文件操作

读取

open()函数，用于打开目标文件，有三个参数

  1. 文件路径，可以是绝对\ 相对路径
  2. 表示读取模式，可以取”r”或”w”，表示读和写;”a”表示附加内容，会在原文件后面跟上新内容
  3. encoding = “…” 指定以什么编码读取文件

read()函数，用于读取目标文件内容，默认情况下会阅读全部内容并以字符串返回文件内容，同一文件第二次调用read()时就会返回空，目标文件最好是txt格式

f = open("./data.txt","r",encoding="utf-8")
print(f.read(10))      #读第1到10字节的文件内容

readline()函数，一次读取一行，常与while循环搭配使用

f = open("./data.txt","r",encoding="utf-8")
line = f.readline()  #读第一行
while line != "":    #判断当前行是否为空
    print(line)       #不为空就打印当前行
    line = f.readline()   #读取下一行
f.close()   #释放资源

readlines()，读取全部内容，并返回每行组成的字符串列表，常与for循环搭配使用

f = open("./data.txt","r",encoding="utf-8")
lines = f.readlines()     #把每行内容存储到列表里
for line in lines:        #遍历每行内容
    print(line)
f.close()

读取完以后要用close()释放资源，也可以在代码前加上with ,后面跟上as f: ,就不需要再close，但原代码要缩进

with f = open("./data.txt","r",encoding="utf-8") as f:
	lines = f.readlines()     #把每行内容存储到列表里
	for line in lines:        #遍历每行内容
    	print(line)

写入

“w”模式在写以前会把原文件清空，谨慎使用

write()函数可以追加内容，默认不会换行

open()的模式要为”w”或”a”,如果原文件不存在Python会创建一个新文件，这两种模式下均不能读取文件内容；要同时进行读和写，就需要把模式设为”r+”，且write的内容会追加在原文件后

with open("./poem.txt","w",encoding="utf-8") as f:
    f.write("床前明月光，疑是地上霜。\n举头望明月，低头思故乡。\n")
    
    
with open("./poem.txt","a",encoding="utf-8") as f:
    f.write("hello,world")

异常处理

通过try/except语句捕捉异常，try后的代码块放容易报错的代码，except后跟异常的名字

except后跟明确的异常类型，如果不知道异常类型就什么都不写

else后跟没有错误时会运行的代码

finally后的内容不管程序错误与否都会执行

try:
    user_weight = float(input("输入体重:"))
    user_height = float(input("输入身高:"))
    user_BMI = user_weight / user_height ** 2
except ValueError:
    print("输入不合理数字")
except ZeroDivisionError:
    print("身高不能为0")
except:
    print("发生未知错误")
else:
    print("您的BMi值为："+ str(user_BMI))
finally:
    print("程序结束运行")

unittest库，用于单元测试，即对软件中最小的可测试单元进行验证，比如验证某函数输出是否符合预期。

import unittest
from my_calculator import my_adder

    def test_positive_with_positive(self):

在测试类下面写测试用例，每个测试用例是一个方法，要注意测试用例必须以test开头Python才能识别。

import unittest
from my_calculator import my_adder

class TestMyAdder(unittest.TestCase):
    def test_positive_with_positive(self):
        self.assertEqual(my_adder(5,3),8)
        
    def test_nagative_with_positive(self):
        self.assertEqual(my_adder(-5,3),-2)

为防止assert中断整个程序运行，可以使用assertEqual()方法，括号内跟布尔表达式。

在终端使用python -m unittest，python就会自动执行测试用例。

测试结果有几个点就代表有几个测试用例，点代表测试通过，F表示未通过。

函数可以传入另一个函数作为参数使用，把函数作为参数的函数叫高阶函数，作为参数的函数直接用函数名，不需要跟括号和参数。

def calculate_square(num):

def calculate_plus_10(num):

匿名函数不需要指定名称，也可称为Lambda，格式如下：

lambda num1,num2 : num1+num2
关键字   函数参数    函数返回的表达式

匿名函数可作为高阶函数的参数传入，也可直接调用，格式如下：

(lambda num1,num2: num1+num2)(2,3)

匿名函数后无法跟多个语句或表达式，只能用于比较简单的场景。
